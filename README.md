# ppds-final

This is the final interface for Projects in Programming and Data Science with Prof. Panos Ipeirotis.

##### To install:

cd into the project folder<br>

`virtualenv venv`<br>
`pip3 install -r requirements.txt`<br>

##### to run the app:

`source venv/bin/activate`<br>
`python3 app.py`
